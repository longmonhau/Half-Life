Half-Life 博客系统

  • 演示：http://longmonhau.com

  • 联系作者：<longmon.hau@gmail.com>

Requirement

  • PHP5.5+

  • fileinfo php extension

  • PDO PHP extension 及相应的数据库扩展

  • 基于LAMP或LNMP，可选Redis作缓存,建议安装在unix系列主机

  • WEB服务器必须支持URL重写

  • Composer 管理依赖

#Apache .htaccess 例子 其他请自行Google
RewriteEngine on
RewriteCond %{REQUEST_FILENAME} !-d
RewriteCond %{REQUEST_FILENAME} !-f
RewriteRule ^(.*)$ /index.php/$1 [L]

Install

  1. 编辑install.php，修改相应的选项

  2. 运行install.php，如果一切正常会输出相关信息

鸣谢

  • nikic/fast-route 一个快速路由器

  • symfony/http-foundation symfony出品的组件

  • twig 模板引擎

  • PHPMailer 邮件库

  • Laravel Eloquent Laravel的ORM

  • monolog 日志

  • codeguy/upload 文件上传组件

  • Jquery

  • mousewheel 基于Jquery的滚轮事件插件

  • dropzone 超级好用的JS上传库
